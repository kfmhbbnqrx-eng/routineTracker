# routineTracker & Calendar

Routine tracker following my daily input of tasks and habits. Monitoring my consumption on what I prioritize and where I need to focus.

## 🌟 Key Features

* **Two-Tier Routine Management:** Define a persistent library of standard master activities, then dynamically build out daily chronological schedules utilizing clear standard ranges.
* **Intuitive Execution Tracking:** Features inline status toggling with active visual tracking states (`checked`, dynamic badge generation) to keep today's objectives isolated before final logging phases.
* **Persistent Local Storage Architecture:** Securely manages data states using the browser's `localStorage` matrix API across application resets or accidental reloads without external server round-trips.
* **Dynamic History Calendar:** Visualizes historical routine distribution patterns with multi-month navigational toggles and full granular task breakdowns upon item interaction.
* **Time Duration Analytics Engine:** An active filtering framework that automatically normalizes time inputs, resolves complex overnight duration boundary matrix edge-cases safely, and tracks core productivity trends across relative weekly/monthly constraints.
